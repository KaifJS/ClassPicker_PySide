# 智能抽号系统 更新日志

## v3.0.1 (2025-05-24)

### ✨ 新增
- 程序启动时自动在桌面创建快捷方式（绕过冰点还原）
- 抽取人数不足时弹出确认对话框，用户可选择是否重置

### 🐛 修复
- 修复抽取动画开关无效的问题（默认不播放动画）
- 修复字体设置仅能手动输入的问题，现提供常用字体下拉选择
- 修复开机自启动参数错误（改用 win32com 直接创建快捷方式）

### ⚙️ 优化
- 安装包支持自动检测 D 盘，优先安装到 `D:\SmartDrawSystem`
- 更新安装脚本不再覆盖用户已有的存档和配置文件

## v3.0.0 (2025-05-20)

- 首个 PySide6 美化版发布
- 支持不重复抽取（预随机顺序）、真随机、可重复+排除三种模式
- 增加记忆存档、开发者命令（作弊指令）、抽号动画、背景音乐等功能




# 智能抽号系统 更新日志

## v3.0.2 Beta (2026-05-25)

### ✨ 新增
- 娱乐功能面板：支持选择音乐文件夹，播放列表管理
- 音乐播放模式：顺序、随机、单曲、单曲循环
- 音量独立控制与实时显示
- 开机自启动路径锁定：强制使用安装目录，彻底解决冰点还原下 C 盘快捷方式丢失问题
- 程序启动时自动重建桌面快捷方式（指向 D 盘安装目录）

### 🐛 修复
- 可重复+排除模式下学生总数异常归零且无法恢复的严重 Bug（引入独立剩余池 `available_students`）
- 字体设置“慢半拍”：修改后立即生效，无需重启程序
- 背景音乐设置缺失问题（现已集成到娱乐功能中）
- 同时勾选“允许重复抽取”和“抽取后排除”时，重置功能正常恢复名单
- 抽取人数不足时弹窗确认逻辑优化

### ⚙️ 优化
- 安装脚本：版本号升至 3.0.2，排除用户数据（saves、app_settings.json）防止覆盖
- 内存与性能优化：音乐播放轮询改用 QTimer，减少 CPU 占用

### 📝 备注
- 若已安装旧版，建议备份 `saves` 和 `app_settings.json` 后覆盖安装
- 首次运行新版会自动记录安装路径到配置文件
