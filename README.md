#🎲 智能抽号系统 v3.0 Beta (PySide6 美化版)

[![Python](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/)
[![PySide6](https://img.shields.io/badge/PySide6-6.6+-green.svg)](https://doc.qt.io/qtforpython-6/)
[![License](https://img.shields.io/badge/license-MIT-yellow.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey)]()

一个基于 PySide6开发的现代化抽号工具，专为课堂提问、随机点名、活动抽奖等场景设计。拥有精美的深色界面、灵活的学生管理、强大的作弊指令、记忆存档、抽号动画等实用功能。

> 本项目是 [Tkinter 原版](https://github.com/KaifrJS/KAIF) 的美化重构版，界面更现代，功能更丰富。

✨ 主要特性

- 🎯 **随机抽取** – 支持“不重复（预随机顺序）”“真随机（可重复）”“可重复+排除”三种模式
- 👥 **学生管理** – 卡片式展示，增删改成员，默认内置55人名单
- 💾 **记忆存档** – 自动/手动保存随机排列顺序，可随时加载历史状态
- 🎬 **抽号动画** – 可开关的数字浮动动画，可调节时长和跳动速度
- 🎵 **背景音乐** – 支持播放本地音乐文件（mp3/wav/ogg等）
- 🖥️ **开发者控制台** – 内置作弊指令（固定位置、交换、移动、洗牌、查看排列等）
- 📊 **统计看板** – 抽取次数、频率排行、剩余人数一目了然，支持导出
- 🔘 **开机自启动** – Windows 下一键设置开机自动运行（无需 winshell，稳定可靠）
- 🎨 **字体定制** – 实时调整结果字体大小、族型、颜色
- 🌙 **深色主题** – 护眼且现代的暗色界面

环境要求
- Python 3.8 或更高版本
- Windows / macOS / Linux
